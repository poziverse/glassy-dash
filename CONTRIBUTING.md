# Contributing

Welcome to the **React Glass Keep** project!

We have recently updated our architecture to support dynamic theming ("Holo-Glass" aesthetic).
Please review the documentation in `docs/` before making changes.

## Documentation Index
- [**Architecture & State**](docs/ARCHITECTURE_AND_STATE.md): Core logic, state management, and asset pipelines.
- [**Development Context**](docs/DEVELOPMENT_CONTEXT.md): Recent changes, current "fragile" areas, and how to add new themes/assets.
- [**Theming Feature**](FEATURE_THEMING.md): Specifics on the Vibe/Preset system.

## Quick Start
1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Access app at: `http://localhost:5173`
